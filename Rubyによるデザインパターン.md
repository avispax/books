# 第 3 章 Template Method

静的型付けの同名デザパタとの違いは、スーパークラスに abstract 等を書かずにオーバーライドすること。  
Ruby のサブクラスでは必要なメソッドのみを書けばよいので、よくある「あー、スーパーのこれ、使わないけど無いと怒られるから空っぽのメソッド書いとくか」ってのが不要。  
挙動とかはテストで担保すること。

# 第 4 章 Strategy
